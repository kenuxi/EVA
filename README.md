# Dash_iris_demo
This code shows how to create interactive plots using dash using iris dataset
fetched from remote source.
# Installing
Create new python3 virtual environment:
```
python3 -m venv venv
```
and source it:
```
source venv/bin/activate
```  
Install libraries:
```
pip install -r requirements.txt
```
Run example:
```
python iris_app.py
```
